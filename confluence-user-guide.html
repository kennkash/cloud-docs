Troubleshooting

Common issues and their solutions to help you resolve problems quickly.

Connection Issues

Symptoms

  • "Unable to connect to server" error message
  • Application freezes during startup
  • Features not loading properly

Solutions

1

Check Network Connection

Ensure your device has an active internet connection. Try loading a website in your browser to verify.

2

Verify Server Status

Check our status page to see if there are any ongoing service disruptions.

3

Restart the Application

Close and reopen the application. If issues persist, try restarting your computer.

If you're using a VPN, ensure it's not blocking the required ports for our application (ports 443 and 8080).

Missing Data Issues

Symptoms

  • Reports showing incomplete data
  • Recently added items not appearing
  • Dashboard widgets displaying "No Data Available"

Solutions

1

Check Filters and Date Ranges

Verify that your filters aren't excluding the data you're looking for. Reset filters to see all data.

2

Refresh Data

Use the refresh button to reload data from the server. This can resolve caching issues.

3

Check Permissions

Ensure you have the necessary permissions to view the data you're trying to access.

Frequently Asked Questions

How do I reset my password?

If you need to reset your password, follow these steps:

  1. Click on the "Forgot Password" link on the login screen
  2. Enter the email address associated with your account
  3. Check your email for a password reset link
  4. Click the link and follow the instructions to create a new password

Password reset links expire after 24 hours for security reasons.

Can I export my data to Excel?

Yes, you can export data from most areas of the application:

  1. Navigate to the page containing the data you want to export
  2. Look for the "Export" or "Download" button (usually in the top right)
  3. Select "Excel" from the format options
  4. Choose whether to export all data or only the current view
  5. Click "Export" to download the Excel file

To export specific columns or filtered data, apply filters before exporting.

How can I add multiple users at once?

For bulk user creation, you can use the CSV import feature:

  1. Go to User Management > Users
  2. Click the "Import Users" button
  3. Download the CSV template provided
  4. Fill in the template with user information
  5. Upload the completed CSV file
  6. Review the import preview and confirm
Example CSV format:
email,firstName,lastName,role,department
user1@example.com,John,Doe,Editor,Marketing
user2@example.com,Jane,Smith,Viewer,Sales
